Creating a Product Assignment

There are 3 scripts to the right. The last two are primarily for viewing purposes, but like the others before it, they can be processed if you are interested in playing around with them.

Now let's focus on the task at hand. Make sure you are on the create.js script before you continue. As you can see in the assignment variable, There are 6 different values. We will only be using 3 of the 6 to create the assignment. To learn more about each of these values, review the documentation.

Let's start by removing the values we don't need. Remove ReplenishmentPriceScheduleID, UserGroupID, and BuyerID. Now that you have the three values you need, go ahead and fill them out with the ID's for each of these. If you forgot the ID's of any of these, go back to the necessary class to recreate one.
