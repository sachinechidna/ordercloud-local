What You Need To Know

If you followed the steps correctly, you have completed everything you need to do in order to accomplish the task of giving a buyer-user access to a product. Now is the final test to see if all the work you have done has been completed correctly. A few high-level concepts to remember as you move on. First, a product will only ever be availabe to a user if that product has a price schedule assigned to both that product and the user you are targeting. Second, a product can only be made accessible to a user through a category. There is absolutely no possible way to assign a user/userGroup/buyer directly to a product. It has to go through a category and category product assignment.
