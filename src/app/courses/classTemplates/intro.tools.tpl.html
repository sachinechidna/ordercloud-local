The OrderCloud Seed

The term "seed" signifies this is the starting point for all OrderCloud applications created through the OrderCloud Development Center. It was crafted using the most advanced web development tools and perfected over years of rapid development in the B2B world.

AngularJS MVW Framework

The OrderCloud Seed runs on AngularJS, a powerful front-end tool created by Google, which provides an environment that is incredibly expressive, readable, and quick to develop. It's extensible toolset can be leveraged in part or in whole alongside other libraries. Choosing AngularJS for our first Seed was an easy decision because it helped us provide our clients with the most flexible Front-End Framework and the fastest development workflow.

It is highly recommended (if you haven't already) that you familiarize yourself with Angular and it's concepts. We will cover some of the features that pertain to the OrderCloud AngularJS SDK (Software Development Kit) in the next class; however, learning everything about Angular requires education beyond the scope of these courses.

Developer Tools
The following tools help the developer work efficiently with the Seed in a way that is compatible with the Development Center.

Node Package Manager (NPM) Installation of Build tools.

NPM is used by the OrderCloud seed to install all of the necessary packages for Building and Compiling your application source code. In the unlikely scenario a developer wants to alter or add to the Ordercloud Build - they will use NPM to install a new package and when done correctly (saving the install to your package.json), the Dev Center will alter it's Build process as well.

When a developer first opens an OrderCloud Seed, running $npm install is the first step toward development.

Bower Front-End package management

Bower is used to install all of the OrderCloud seeds Front-End modules (listed below) - including AngularJS. The developer will run a $bower install in the console after they complete the NPM install process. This will place each package beneath a brand new "vendor" directory which will be referred to during the Build process.

Gulp A streaming build system

Gulp is a toolkit that will help OrderCloud developers automate painful or time-consuming tasks in their development workflow. For web development it can help by doing CSS preprocessing, JS transpiling, minification, live reloading, and much more. Integrations are built into all major IDEs.

In the Seed, Gulp additionally takes care of cross-browser CSS, Angular Injection patterns, and even improves performance by compiling all project HTML files into one JS file that stores them in the templateCache.

The Front-End
The Seed comes bundled with libraries that replace default AngularJS features and others that enhance Angular as it exists.

AngularUI Router Replaces Angular's ngRoute

AngularUI Router is a routing framework for AngularJS, which allows you to organize the parts of your interface into a state machine. Unlike the $route service in the Angular ngRoute module, which is organized around URL routes, UI-Router is organized around states, which may optionally have routes, as well as other behavior, attached.

States are bound to named, nested and parallel views, allowing you to powerfully manage your application's interface.

Bootstrap Design Framework of choice

Bootstrap is a sleek, intuitive, and powerful front-end framework for faster and easier web development, created by Mark Otto and Jacob Thornton, and maintained by the core team with the massive support and involvement of the community.

Bootstrap was chosen due it's high popularity in the developer community and the fact that it runs on CSS Preprocessors; these allow for incredibly quick customization and themeing.

AngularUI Bootstrap UI Library compatible with Bootstrap

Native AngularJS (Angular) directives for Bootstrap. Smaller footprint (20kB gzipped), no 3rd party JS dependencies (jQuery, bootstrap JS) required. Each JS Componenet described in the Bootstrap documentation is available as an Angular directive through this fantastic UI library. All the most common UI components are accounted for and are incredibly easy to implement.

JQuery Javascript Library (dependency of AngularUI Bootstrap)

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

Angular Auto-Validate Automatic Form Validation

An automatic validation module for AngularJS which gets rid of excess html in favour of dynamic element modification while still using the native angular validation mechanism. It gets away from having to specify validation messages upfront in the html and aims to drastically reduce markup complexity.

FontAwesome Font icons

In a single collection, Font Awesome is a pictographic language of nearly 600 web-related actions. Font Awesome provides scalable vector icons that can instantly be customized -- size, color, drop shadow, and anything that can be done with the power of CSS.

The cheatsheet provides a comprehensive list of the available icons.
